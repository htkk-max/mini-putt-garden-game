# 迷你高尔夫花园 (Mini Putt Garden)

## 游戏介绍

### 游戏背景
欢迎来到《迷你高尔夫花园》的休闲世界！这是一款充满乐趣和挑战的高尔夫小游戏，你将在美丽的花园环境中体验高尔夫的乐趣，通过精准的击球技巧将球打入洞中。

### 游戏概述
《迷你高尔夫花园》是一款基于HTML5和JavaScript开发的休闲高尔夫游戏，玩家需要通过拖拽控制击球力度和方向，将球打入洞中。游戏包含多个精心设计的关卡，每个关卡都有不同的地形和障碍物，考验你的精准度和策略思维。

## 游戏玩法详解

### 基本操作
- **PC端**：
  - 鼠标拖拽：在高尔夫球周围拖拽鼠标，控制击球的力度和方向
  - 释放鼠标：击球，将球按照预设的方向和力度打出
  - 屏幕上方按钮：
    - 暂停：暂停游戏
    - 静音：开启/关闭游戏音效
  
- **移动端**：
  - 手指拖拽：在高尔夫球周围拖拽手指，控制击球的力度和方向
  - 释放手指：击球，将球按照预设的方向和力度打出
  - 屏幕上方按钮：
    - 暂停：暂停游戏
    - 静音：开启/关闭游戏音效

### 游戏目标
- **核心目标**：用最少的杆数将高尔夫球打入洞中，完成每个关卡
- **次要目标**：
  - 收集关卡中的宝石，获得额外分数
  - 尽可能用少于标准杆数完成关卡
  - 解锁并完成所有关卡

## 游戏规则详解

### 核心玩法
- **击球机制**：通过拖拽控制击球力度和方向，拖拽距离越长，击球力度越大
- **标准杆**：每个关卡都有预设的标准杆数，用少于或等于标准杆数完成关卡可以获得更高分数
- **障碍物**：游戏中包含各种障碍物，如水池、沙地等，球进入这些区域可能会被重置或受到其他影响
- **传送门**：某些关卡中可能包含传送门，可以将球传送到其他位置

### 得分系统
- **基础得分**：完成关卡获得基础分数
- **杆数奖励**：用少于标准杆数完成关卡获得额外奖励
- **宝石收集**：收集关卡中的宝石获得额外分数

## 如何开始游戏
1. 在浏览器中打开 `index.html` 文件
2. 等待游戏资源加载完成
3. 点击游戏界面开始按钮
4. 选择游戏课程
5. 点击开始游戏
6. 在高尔夫球周围拖拽鼠标/手指，控制击球力度和方向
7. 释放鼠标/手指击球，将球打入洞中

## 游戏特色
- **精美画面**：美丽的花园背景，精心设计的关卡场景
- **简单易上手**：直观的拖拽操作，适合所有年龄段玩家
- **多样化关卡**：多个精心设计的关卡，每个关卡都有不同的挑战
- **物理引擎**：真实的物理碰撞效果，提供流畅的游戏体验
- **收集要素**：每个关卡中隐藏的宝石，增加游戏趣味性
- **跨平台支持**：支持PC和移动设备，随时随地享受游戏乐趣

## 技术要求
- **PC端**：
  - 支持HTML5的现代浏览器（Chrome、Firefox、Safari、Edge等）
  - 分辨率建议1024x768或更高
  - 基本的鼠标或触摸输入设备
  
- **移动端**：
  - 推荐使用Chrome或Safari浏览器
  - 屏幕尺寸3.5英寸以上
  - iOS或Android操作系统
  - 支持触摸操作

## 游戏提示
1. 观察关卡布局，规划最佳击球路线
2. 控制击球力度，避免用力过猛导致球出界
3. 注意障碍物的位置，合理利用反弹
4. 收集关卡中的宝石，获得额外分数
5. 尽量用少于标准杆数完成关卡，提高得分
6. 如果球进入困难区域，可以等待球静止后重新开始击球
7. 适当调整击球角度，利用墙壁反弹来绕过障碍物

## 故障排除
- 游戏加载缓慢：尝试清理浏览器缓存或更换网络环境
- 画面卡顿：降低浏览器标签页数量或关闭后台程序
- 操作不响应：检查鼠标或触摸屏是否正常工作
- 音效不播放：检查设备音量设置或点击游戏中的静音按钮
